# 家教系统前端项目

## 项目简介
家教系统前端项目，基于Vue 3 + TypeScript + Vite + Element Plus + Pinia开发，提供学生、教师和管理员三个角色的功能界面。

## 技术栈
- Vue 3
- TypeScript
- Vite
- Element Plus
- Pinia

## 环境要求
- Node.js 18+

## 安装与运行

### 安装依赖
```bash
pnpm install
```

### 开发运行
```bash
pnpm dev
```

### 构建
```bash
pnpm build
```

### 代码检查
```bash
pnpm lint
```
